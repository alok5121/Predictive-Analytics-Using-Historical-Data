# 📈 Predictive Analytics Using Historical Data

## Project Overview

This project leverages historical sales data to build a predictive analytics solution capable of forecasting future sales trends. The objective is to analyze past sales patterns, develop a machine learning model, and provide sales predictions through an interactive Streamlit dashboard.

---

## Objectives

* Analyze historical sales records
* Perform data cleaning and preprocessing
* Conduct exploratory data analysis (EDA)
* Build a predictive machine learning model
* Forecast future sales
* Visualize business insights
* Deploy an interactive dashboard

---

## Dataset

The dataset contains historical sales transactions with attributes such as:

* Order Date
* Customer Information
* Product Information
* Region
* Sales
